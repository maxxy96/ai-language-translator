# ai-language-translator
AI-powered multilingual translator web application with 100+ language support, speech synthesis, translation history, tone customization, and responsive UI built using HTML, CSS, JavaScript, and Node.js.

# AI-Based Language Translator

**AI-Based Language Translator — Auriseg Full Stack Internship Assignment**

---

## Project Overview

This project is a simple Full Stack Language Translator application developed as part of the Auriseg Full Stack Developer Internship assessment.

The main purpose of this project is to translate text from one language to another in real time using an external translation API.
